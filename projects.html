recent projects

  • brain tumor segmentation from mri

    deep learning project focused on automatic brain tumor segmentation from mri scans, as part of the mti865 course on medical imaging. the goal was to assist clinicians by producing accurate tumor masks that distinguish pathological regions from healthy tissue.

    i implemented u-net–based convolutional neural networks adapted to 2d medical slices, with dedicated pre-processing, loss functions (dice and weighted cross-entropy), and evaluation metrics for segmentation quality.

    github link

  • early stroke prediction with big data

    predictive analytics project aiming to estimate the probability of stroke for patients using demographic and medical features, in the context of the mti850 course on big data analytics.

    i built end-to-end pyspark pipelines for large-scale data processing, feature engineering, and model training, comparing several statistical models with metrics such as recall to prioritize early detection.

    github link

  • species recognition application database

    professional project conducted with the muséum national d'histoire naturelle de paris (mnhn), aimed at creating a comprehensive training database for species recognition.

    i created a training database containing over 5,000 frog images with associated metadata, developed automated web scraping scripts for data collection, and built metadata-based annotation systems. this work was conducted in collaboration with a team of researchers from the museum and the start-up ai.nature.

  • virtual reality environment for spatial audio research

    research project for the mti860 course, developing a virtual reality environment in unity to evaluate sound localization accuracy and the weighting of sensory inputs in audio identification.

    the vr environment was designed to test sound localization precision under various challenging conditions including environmental noise, incoherent sound sources, varying speeds, and other distractions. this work provides insights into how human perception adapts to complex auditory scenarios in immersive virtual spaces.

  • nutritional data analysis & clustering

    data analysis project for the sy09 course, exploring a nutritional dataset to understand relationships between nutrients and to group foods into coherent families.

    i applied pca and mds for dimensionality reduction and visualization, then performed hierarchical clustering (cah) to identify meaningful food clusters, using python, scikit-learn, and standard data science tooling.

    github link

  • dfs in common lisp – search in a dynamic world

    academic project for the ia01 course, implementing a depth-first search (dfs) algorithm in common lisp in a harry potter–inspired world where an agent must explore a map, collect items, and destroy horcrux-like targets.

    the work focused on handling a dynamic state space, inventory management, constraints, and limited-depth dfs, with an extended two-agent mode (harry vs. voldemort) introducing interactive behavior.
